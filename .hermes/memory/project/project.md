# Project: $repo

## Goal
Active development with br/bv-powered workflow. Cron agents work beads through the full lifecycle: brainstorm → create → plan → ship → verify → review → pr.

## Success Criteria
1. Open beads are picked up and completed
2. Quality minimums met (PRD >= 600, plan >= 600, tasks >= 400, review >= 200)
3. All work on feature branches, never main
4. PRs created for all completed work
